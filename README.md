# 🕌 دار روضة القرآن — نظام الإدارة المتكامل

نظام مجاني لإدارة دار تحفيظ القرآن: إدخال سهل للسكرتير، تحليلات، مالية بالساعة،
تقاويم وتقارير PDF احترافية، وتقييمات للمعلمين وأولياء الأمور — بواجهة عربية RTL
بهوية الدار (تركواز + ذهبي).

- **الواجهة:** Streamlit (Python) — تُستضاف مجانًا على Streamlit Community Cloud.
- **قاعدة البيانات:** Google Sheets (مخطط علائقي: محفظون/أولياء أمور/طلاب/تسجيلات/حصص).
- **المستندات:** PDF عربي عبر reportlab + arabic-reshaper + python-bidi.

## التشغيل السريع

```powershell
pip install -r requirements.txt
streamlit run app.py        # الدخول: admin / dar2026
```

للإعداد الكامل (الحفظ في Google Sheets، النماذج، النشر، الترحيل): راجع **[SETUP.md](SETUP.md)**.

## المميزات الرئيسية
- ⚡ **توليد حصص الشهر بضغطة** من النمط الأسبوعي — يلغي إعادة الإدخال الشهري.
- 🔁 ترحيل شهري جماعي لكل التسجيلات النشطة.
- 💰 رواتب بالساعة + رسوم فودافون كاش (×1.01) + تقريب لأعلى لأقرب 5 ج.م.
- 📄 تقاويم وتقارير شهرية PDF للطلاب والمحفظين.
- ⭐ تقييم بعد كل حصة + رضا أولياء الأمور (نماذج Google) وملخصاتها.
